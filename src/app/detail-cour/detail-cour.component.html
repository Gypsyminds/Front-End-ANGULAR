  • Une initiation au monde de l'algorithmique et du développement web

  • Gérer les projets via Git/Trello/Jira

  • Commencer avec les framework de développement web en frontend React/Angular
  • Développez votre Backend avec express.js
  • S'initier au monde des bases de données (mongoDb)
  • Déployer votre application web sur un serveur
